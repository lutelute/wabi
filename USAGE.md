# 侘び — 使い方ガイド

## はじめに

**侘び (wabi)** は、日々のルーティンを静かに管理・実行するためのアプリです。枯山水の砂紋をモチーフにしたビジュアルで、タスクの進捗を直感的に感じ取れます。

---

## 画面構成

### サイドバー（左）

- **ルーティン一覧**: 登録したルーティンが並ぶ
- **ひとこと**: 時間帯に応じたソフトな提案（概念ルーティン）
- **カレンダー**: 月間の活動状況を一覧表示

### メインエリア（右）

上から順に以下の要素が配置されます：

1. **時計 / Now Focus** — 現在時刻と次のタスクを表示。タイマー中はカウントダウン表示に切り替わる
2. **枯山水ビジュアル** — 今日・今週・今月の3つの庭。タスクを完了するほど砂紋が整っていく
3. **心のメモ** — 自由記述エリア。気づきや手放したことを書く
4. **今日を閉じる** — 一日の区切りをつけるボタン
5. **侘び（左列）** — アクションチェックリスト。今日やるタスク一覧
6. **寂び（右列）** — チェックインパネル。体と心の状態を記録

## 基本ワークフロー

### 1. ルーティンを作る（編集モード）

画面上部の「編集 / 実行」トグルで **編集モード** に切り替え。

ルーティンはMarkdownライクな記法で定義します：

```
## 朝のセットアップ
- 06:00 ストレッチ 10min *2
- 06:15 瞑想 15min *3 @mental
- 06:30 朝食

## 集中作業
- 09:00 ディープワーク 90min *5
- 10:30 休憩 10min @rest
```

| 記法 | 意味 |
|------|------|
| `## タイトル` | フェーズ（グループ）の区切り |
| `- タスク名` | タスク項目 |
| `06:00` | 開始予定時刻（任意） |
| `10min` | 所要時間（任意、タイマー連動） |
| `*3` | 重み（1〜5、体力消費に影響） |
| `@mental` | 内面ワーク（振り返りが必須になる） |
| `@rest` | 休息タスク（体力回復トリガー） |

### 2. 今日のアクションを組む（実行モード）

サイドバーのルーティンをクリックして展開し、今日やりたいアイテムを選択。

- **個別追加**: アイテムをクリックでON/OFF
- **一括追加**: ルーティン名の横の「全入」ボタン
- **ホバーポップアップ**: ルーティン名にカーソルを合わせると詳細表示。ここからも個別追加可能

複数のルーティンから自由に組み合わせられます。

### 3. タスクを実行する

チェックリストの各タスクを完了していきます：

- **通常タスク**: チェックボックスをクリックで完了
- **タイマー付きタスク**: クリックでタイマー開始。終了後に自動で完了処理。`skip` で即完了も可能
- **@mental タスク**: 長押しで開始。完了時に振り返りコメントの入力が求められる
- **完了後**: 気分ボタン（重い / もやもや / ふつう / 穏やか / 軽い）でタスクごとの感触を記録
- **ひとこと**: タスクにカーソルを合わせると `+ ひとこと` が表示。短いメモを残せる
- **ダブルクリック**: タスク名をダブルクリックでリネーム
- **重み調整**: 「重み」ボタンでスライダー表示。タスクの体力消費を調整

### 4. チェックイン（寂び）

右カラムの「寂び」パネルで、定期的に自分の状態を記録します。

#### 4軸ゲージ
体と心の2カテゴリ、計4軸をスライダーで記録：

| 軸 | カテゴリ | 左端 | 右端 |
|----|----------|------|------|
| 体温 | 体 | 冷 | 熱 |
| 体力 | 体 | 満 | 尽 |
| 淀 | 心 | 濁 | 澄 |
| 波 | 心 | 凪 | 荒 |

- **体力の上限**: 前回のチェックイン値がキャップになる（基本的に減る一方）
- **休息ボタン**: 押すと上限+30まで回復可能。`@rest` タスクの完了でも自動発動

#### 気分ボタン
重い / もやもや / ふつう / 穏やか / 軽い の5段階。タイムラインで変化を追える。

#### 気持ちタグ
プリセットタグ（疲れた、すっきり、集中できた等）から選択。カスタムタグも追加可能。

#### 時系列グラフ
2回以上チェックインすると、各軸の推移がミニグラフで表示されます。

### 5. 一日を閉じる

「今日を閉じる」ボタンで一日を区切ります。閉じた後も「再開する」で復帰可能。

## 枯山水ビジュアライゼーション

3つの庭が表示されます：

- **今日**: 今日のアクション達成率に連動（大きい庭）
- **今週**: 今週の平均達成率
- **今月**: 今月の平均達成率

未着手ではパーティクルがノイズ状に散らばり、進捗が上がるにつれ同心円の砂紋へ収束。100%達成で砂紋がゆっくり呼吸するアニメーション。

---

## エクスポート

### Export ボタン（ヘッダー右）
一日の記録をMarkdown形式でクリップボードにコピー。ObsidianなどのノートアプリにそのままPaste可能。

### 寂びの記録コピー
チェックインパネル下部の「全記録をコピー」で、チェックイン履歴をまとめてコピー。AIに貼り付けて振り返り相談にも使えます。

### Obsidian連携（デスクトップ版のみ）
設定画面でObsidianのデイリーノートフォルダを指定すると、データ保存時に自動で `YYYY-MM-DD.md` を書き出し。YAML frontmatter付き。

## 設定

設定画面（歯車アイコン / メニュー > 設定）で以下を調整：

| カテゴリ | 項目 | 説明 |
|----------|------|------|
| タイマー | 完了時に通知 | タイマー終了時のシステム通知 |
| 体力 | ソフトキャップ | 「よくやった」表示の閾値（50〜100%） |
| 体力 | 重みの範囲 | スライダーの最大値（5 or 10） |
| 体力 | デフォルトの重み | 新規タスクの初期ウェイト |
| 枯山水 | 砂粒の密度 | パーティクル数（少/標準/多） |
| 枯山水 | 砂紋リング数 | 同心円の本数（3〜15） |
| ガイド | 概念ルーティン | 時間帯に応じた提案の表示ON/OFF |
| ガイド | リマインダー音 | リマインダーのサウンド通知 |
| データ | 実行履歴クリア | 進捗データのリセット（ルーティン定義は保持） |
| バックアップ | エクスポート/インポート | 全データのJSON保存・復元 |
| Obsidian | デイリーノートフォルダ | 自動書き出し先の設定 |
| アカウント | ログイン | Supabase認証によるクラウド同期 |

## Tips

- **ルーティンの色分け**: サイドバーでルーティン名にホバーするとカラーパレットが表示。マーカーハイライトで視認性UP
- **ルーティンのメモ**: ホバーポップアップ内、またはエディタ下部にメモ欄あり
- **ルーティン名の変更**: サイドバーでダブルクリック
- **複数ルーティンの組み合わせ**: 異なるルーティンからアイテムを自由にピック可能
- **アイテムの複数追加**: ホバーポップアップの `+` ボタンで同じアイテムを複数回追加できる
- **データは日ごとにリセット**: 毎日が新しい庭。意図的なリセットで「無常」を体現
- **モバイル対応**: レスポンシブデザイン。サイドバーはハンバーガーメニューから開閉

## 技術情報

- **デスクトップ**: Electron 40 (macOS)
- **Web**: Vite 7 dev server (`npm run dev:web`)
- **ストレージ**: Electron版は electron-store、Web版は localStorage
- **クラウド同期**: Supabase（オプション、環境変数で有効化）
- **自動更新**: Electron版は electron-updater による自動ダウンロード+インストール
